AI Resume Screening System

Rank candidates automatically using AI

🤖 Machine Learning Model Used

Model

Sentence-BERT (all-MiniLM-L6-v2)

Used to convert text into semantic embeddings.

Algorithm

Cosine Similarity

Measures similarity between resume and job description embeddings.

Skill Matching

Extract skills from JD and resume and compute match percentage.

Final Score

Final Score = 0.7 × Semantic Similarity + 0.3 × Skill Match

⚙️ How the AI Ranking Works

1️⃣ Resume Upload
→
2️⃣ Text Extraction
→
3️⃣ AI Embedding
→
4️⃣ Similarity Calculation
→
5️⃣ Skill Matching
→
6️⃣ Candidate Ranking
